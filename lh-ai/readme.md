# 蓝湖AI助手
## 创建一个chrome本地插件
## 插件的基本功能，提供一个扫描按钮

1. 点击扫描按钮，分析当前页面的元素
    - 元素激活区域只起到统计多少个元素的作用，点击后右侧面板会更新， 主要应该把右侧面板的内容分析出来 。
    - 需要把右侧“样式信息”中的内容提取出来；需要把“代码区域”的内容提取出来
    - 找到右侧面板的元素后，尝试用innerText来提取文本
2. 要分析的元素所在区域：//*[@id="detail_container"]
3. 元素激活区域：//*[@id="detail_container"]/div[2]/div[2]/div[1]/div[3]/div[1]，点击最内侧的元素后，会更新右侧面板
4. 右侧面板区域：//*[@id="detail_container"]/div[5]
5. 右侧面板中有样式信息区域：//*[@id="detail_container"]/div[5]/div[1]/div[3]/div[1]
6. 右侧面板中有代码区域：//*[@id="detail_container"]/div[5]/div[1]/div[3]/div[3]/div/div[2]/pre

7. 将分析过程展示在插件的弹窗中
    - 应该深度遍历元素激活区域中的每个元素，元素点击后，再分析右侧面板中的“样式信息区域”和“代码区域”
    - 元素激活区域，点击后，需要等待右侧更新了
    - 利用innerHTML来获取HTML代码
8. 分析完成后，创建一份json文件
