# Skills 技能目录

本目录包含所有可用的技能集。

## 技能列表

| 技能名称 | 描述 | 路径 |
|----------|------|------|
| **yapi-create** | 从 YAPI 接口文档创建实体类文件 | [yapi-create/](yapi-create/) |
| **svg-color** | 替换 SVG 文件中的指定颜色 | [svg-color/](svg-color/) |
| **mini-route** | 根据文件名称创建文件和路由 | [mini-route/](mini-route/) |
| **template-file-super** | 模板文件生成工具 | [template-file-super/](template-file-super/) |
| **font-size-accessibility** | 字体大小无障碍工具 | [font-size-accessibility/](font-size-accessibility/) |

---

## 使用说明

每个技能目录包含：
- `SKILL.md` - 技能定义文件
- `scripts/` - 技能实现脚本
- `references/` - 参考模板（部分技能）

请参考各技能目录下的 `SKILL.md` 了解具体使用方法。
