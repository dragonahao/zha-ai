# Luca UI 布局滚动组件

要创建滚动容器时，可以根据需要选择合适的组件类型，并参考对应的详细文档进行使用。每个组件都提供了丰富的属性和方法，帮助你轻松实现各种滚动样式和功能。

## 组件列表

| 组件                   | 用途                                               | 详细文档                               |
| ---------------------- | -------------------------------------------------- | -------------------------------------- |
| LucaExpandTopWidget    | 仅顶部滑动容器+底部固定区域+两段式布局             | `references/expand-top-scroller.md`    |
| LucaExpandBottomWidget | 仅底部滑动容器+顶部固定区域+倒置二段式布局         | `references/expand-bottom-scroller.md` |
| LucaExpandCenterWidget | 通用滚动容器+可以配置顶部和底部区域+经典三段式布局 | `references/expand-center-scroller.md` |
